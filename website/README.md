# OpenDraft - Landing Page

Official landing page for [OpenDraft](https://github.com/federicodeponte/opendraft), an AI-powered academic writing framework.

## About

This landing page showcases OpenDraft, a 100% free and open source framework that helps researchers and students write academic theses 10x faster using 14 specialized AI agents.

### Key Features Highlighted

- **14 Specialized AI Agents** - From Scout (research) to Polish (refinement)
- **200M+ Research Papers** - Integrated access to arXiv, Semantic Scholar, PubMed, Google Scholar
- **Multi-LLM Support** - Claude Sonnet 4.5, GPT-4o, Gemini 2.5 Flash
- **Auto Citation Verification** - Fact-checks against CrossRef and arXiv
- **Publication-Ready Exports** - PDF, Word, LaTeX
- **FREE Tier Available** - Gemini Flash free tier (up to 12k words)

## Tech Stack

- **Next.js 14.2** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **shadcn/ui** - Component library
- **Lucide Icons** - Icon system

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
